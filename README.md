# simonebellavia.com

My personal website, using Hugo as a SSG.

- Built with [Hugo](https://gohugo.io/).
- Started from the [Basic](https://github.com/siegerts/hugo-theme-basic) theme. Then customized according to my needs.
- Hosted on [Netlify](https://www.netlify.com/).
## How to run it

Clone the repository. Then run Hugo webserver to see the website locally:

    hugo server
