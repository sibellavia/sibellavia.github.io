Lemon is a fast, modern, general-purpose web server. It is built with performance and easy configuration in mind. Lemon is a single, cross-platform binary with minimal digital footprint. 

Lemon leverages high-quality, performant libraries such as [Hyper](https://github.com/hyperium/hyper) and [Tokio](https://github.com/tokio-rs/tokio).

Lemon is under active development, and it will be for a long time. Enjoy us on our journey.

Lemon is Simone Bellavia's side project. [Consult the license here](https://github.com/sibellavia/lemon/blob/main/LICENSE).
